# API Plant Information 

## App Statement : 
### Creating API that help people to know outdoor and indoor plant around the world and how to take care of them 


## Feature List : 
* get all plant info
* get indoor plants info 
* get outdoor plants info
* Add a new plant
* delete plant info
* update plant info
